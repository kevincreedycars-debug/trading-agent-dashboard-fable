# Workflow Documentation

This directory stores human-readable documentation for each n8n workflow.

Expected files:

- `master_orchestrator.md`
- `eco_events_collector.md`
- `usd_collector.md`
- `eur_collector.md`
- `gold_collector.md`
- `nq_collector.md`
- `btc_collector.md`
- `usd_layer1_agent.md`
- `eur_layer1_agent.md`
- `gold_layer1_agent.md`
- `nq_layer1_agent.md`
- `btc_layer1_agent.md`
- `dashboard_writer.md`

Each workflow document should include:

- purpose
- trigger type
- inputs
- outputs
- Supabase tables touched
- key nodes
- known issues
- last verified status
